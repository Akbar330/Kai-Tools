# Kai Tools 🚀

**Kai Tools** adalah suite keamanan dan intelijen lengkap yang dirancang untuk edukasi *ethical hacking*. Suite ini menggabungkan berbagai teknik dari OSINT hingga eksploitasi dalam satu antarmuka baris perintah (CLI) yang kuat.

## Fitur Utama

- **Kai OSINT**: Pencarian WHOIS, DNS, Geolocation IP, pencarian username sosial media, ekstraksi metadata foto (Exif), dan pelacakan nomor HP.
- **Wireless Auditor**: Analisis keamanan jaringan WiFi di sekitar Anda.
- **Network Security**: Port scanning multi-threaded, MAC Spoofing, dan pencarian basis data kerentanan (CVE).
- **Web Pentesting**: Automated scan untuk Clickjacking, XSS, folder tersembunyi, hingga celah SQL Injection.
- **Exploitation Lab**: Generator payload Reverse Shell untuk Bash, Python, PHP, dan PowerShell.
- **Crypto Suite**: Identifikasi jenis hash dan pemecah sandi (*Hash Cracker*).

## Cara Menggunakan

1. **Persiapan**:
   ```bash
   source venv/bin/activate
   ```

2. **Jalankan Suite**:
   ```bash
   python main.py
   ```
